# Operation log

Append-only change log for this wiki. Each entry records a mutation to the
knowledge base in the form `## [YYYY-MM-DD] <type> | <summary>`. Entries are
never rewritten, only appended, so the log reads as a faithful history of how
this documentation came to be — distinct from the [roadmap](roadmap.md), which
looks forward, and from [releases and versioning](concepts/releases-and-versioning.md),
which tracks the module code the wiki describes.

The wiki itself is a product of the [wiki module](modules/wiki.md); entries
here follow the same append-only discipline that module applies to campaign
knowledge bases. Return to the [Index](index.md) for the full catalog of pages.
